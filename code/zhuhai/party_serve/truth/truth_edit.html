尊敬的书记：
    我局在征收车辆购置税后将向纳税人开具《税收完税证明》和《车辆购置税完税证明》，其中《车辆购置税完税证明》是用于车主注册登记向车管所提交的完税凭证，而《税收完税证明》则用于记账。由于我局面对的纳税人基本上都是最终消费者个人，无需记账，为此，我局决定从2017年2月13日起取消《税收完税证明》的打印，如纳税人有需求可随时申请补打。此举将有效节约纳税人的办税时间，同时也提高前台受理人员的工作效率，切实倡导“两个减负”。
    表现在票证管理方面，购置税自助办税终端机的票证发放安全更有保障。我市共有7台购置税自助办税终端机，分布较散，最远的自助办税点距离我局40多公里，7个自助办税点至我局的距离合计超过100公里，且自助办税点所处位置为交通繁华路段，经常发生交通拥堵。我局3名终端机专职管理员每个工作日需随身携带几百份票证来回奔波超过100公里，长途跋涉中票证安全存在较大隐患。取消打印《税收完税证明》后，各自助办税终端机不用再放置《税收完税证明》，可较大程度地降低票证管理风险,也避免了自助办税终端机打印《税收完税证明》经常出现卡纸的现象，在提高管理员工作效率同时，也实现终端机为前台减压的使用初衷。
